# Test Layout

- `ports/`: compatibility and equivalence tests against external systems such as Boxes.py.
- `integration/`: end-to-end browser-generator workflow tests across domain, fabrication, and export.
- `domain/`: design-specific geometry and model tests.
- `app/`: URL/workbench state and UI-state helper tests.
- `rendering/`: renderer helper tests that can run without a browser screenshot.
